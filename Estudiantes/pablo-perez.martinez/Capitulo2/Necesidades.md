# Necesidades del sistema

A partir de las reuniones llevadas a cabo con el cliente y del análisis del funcionamiento actual de VT, las necesidades del sistema pueden agruparse en 2 módulos:
- Módulo de backend: la lógica y el cálculo de datos a partir de la información disponible se desarrollará en el backend. 
- Módulo de frontend: una vez los cálculos se realicen en el backend con precisión, se integrarán en el frontend facilitando su comprensión mediante herramientas gráficas y tablas comparativas.
