# Necesidades del sistema

A partir de las reuniones mantenidas con el cliente y del análisis del funcionamiento actual de VT, se identificó la necesidad de desarrollar un sistema capaz de procesar y calcular con precisión el OEE y sus indicadores a partir de los datos disponibles, y al mismo tiempo, integrarlos en la interfaz de forma clara e intuitiva para que su interpretación sea sencilla mediante herramientas gráficas y tablas comparativas.
